# 💻 Quiz Interativo para Programadores

Um quiz temático para testar conhecimentos em desenvolvimento web, com visual retro-futurista e feedback em tempo real para cada resposta.

---

## 📁 Estrutura do Projeto

```
quiz-programadores/
├── index.html       # Estrutura e conteúdo do quiz
├── style.css        # Estilização completa (tema dark retro-futurista)
├── script.js        # Lógica de interatividade e verificação
└── python.jpeg      # Imagem usada na pergunta 8
```

---

## 🎯 Funcionalidades

- **8 perguntas** cobrindo HTML, CSS, redes e lógica de programação
- **Tipos de questão variados:** múltipla escolha, texto livre, senha, data, checkbox, upload de arquivo, menu suspenso e identificação de imagem
- **Verificação em tempo real** com feedback visual (verde = correto, vermelho = errado)
- **Barra de progresso** sticky no topo com pontuação ao vivo
- **Modal de resultado final** com animação de pontuação
- **Medidor de força de senha** dinâmico
- **Drag & drop** para upload de arquivo `.html`
- **Efeito de digitação** no título do header
- **Partículas animadas** em canvas no header
- **Toast notifications** para cada interação
- **Tabela de pontuações** e gabarito recolhível

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica do quiz |
| CSS3 | Estilização, animações e layout responsivo |
| JavaScript (ES6+) | Interatividade, verificação e manipulação do DOM |
| Google Fonts | Fontes Orbitron e Rajdhani |
| Canvas API | Partículas animadas no header |
| IntersectionObserver API | Animação de entrada das seções |

---

## 🎨 Design

O projeto adota um tema **retro-futurista** com:

- Paleta **laranja queimado × roxo escuro**
- Fontes `Orbitron` (display) e `Rajdhani` (corpo)
- Fundo escuro com scanlines e gradientes radiais animados
- Cantos angulares decorativos nos cards
- Botões com `clip-path` em chanfro estilo sci-fi

## 📝 Perguntas do Quiz

| # | Tipo | Tema |
|---|---|---|
| 1 | Múltipla escolha | Linguagem de estruturação web |
| 2 | Texto livre | Componentes de URL |
| 3 | Senha | Boas práticas de segurança |
| 4 | Data | História do HTML |
| 5 | Checkbox | Linguagens de programação |
| 6 | Upload | Arquivo HTML válido |

## 📊 Sistema de Pontuação

| Pontuação | Avaliação |
|---|---|
| 0 – 2 | Continue tentando! A prática leva à perfeição. |
| 3 – 4 | Bom começo! Revise os conceitos. |
| 5 – 7 | Muito bom! Seu conhecimento está sólido. |
| 8 | Perfeito! Você domina programação web! |
